CINES OLIVIA
Description

In this project, the dynamic website of a cinema was created taking into account the needs of the administrators as well as the security of the users and the data.

Technologies

☑ PHP
☑ SQL (MARIADB)
☑ HTML | JS | CSS

VERSION 1

In this first version it is dedicated only to the frontend, using only html and css. Try to get as aesthetic a look as possible, but none of the buttons are functional. It is a static and responsive page.

VERSION 2

In this second version it is dedicated only to the backend (my favorite), using JS and PHP. It tries to achieve the greatest functionality and security, dynamic tables, protection against SQL injections, avoiding improper access and redirection only to pages in .php format through the .htacces file. The frontend is completely neglected, since the objective is to develop a functional and reliable backend.

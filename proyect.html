CINES OLIVIA
Description

In this project, a cinema dynamic website was created thinking about the administrators needs, the security of the users and the data as well.

Technologies

☑ PHP
☑ SQL (MARIADB)
☑ HTML | JS | CSS

VERSION 1

In this first version it is dedicated only to the frontend, using only html and css. Try to get as minimalist style as much as possible, but none of the buttons are functional. It is a static and responsive page.

VERSION 2

In this second version it is dedicated to the backend (my favorite by the way), using JS and PHP. It tries to achieve the greatest functionality and security, using dynamic tables, protection against SQL injections, avoiding improper access and redirectioning only to pages in .php format through the .htacces file. In this version, the frontend is completely forgotten since the objective is to develop a functional and reliable backend.
